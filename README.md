# NodeJS-App

A web application using Node.JS.

This application needs the following modules:
- express.js
- jade

Technologies & Techniques used in this application inclues:
- Bootstrap
- Express Http Server
- Jade Templating 
- Modular
- Inheritance in Templating
- Extends (Jade)
- Include (Jade)
- Data Injection
- Blocks (Jade)
- Routing
